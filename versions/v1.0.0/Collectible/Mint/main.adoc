==== Mint

Mint indicates an action of minting a new token/tokens.

===== Interface

[,typescript]
----
interface Mint {
    address?: string;
    id?: string;
    name?: string;
    standard?: string;
    symbol?: string;
    value: string;
    uri?: string;
    parsed_image_url?: string;
}
----
