===== Stake

Stake indicates an action to staking a token/tokens.

===== Interface

[,typescript]
----
interface Stake {
    action: string;
    token: {
        address?: string;
        value: string;
        name?: string;
        symbol?: string;
        decimals?: string;
        standard?: string;
    };
    period?: {
        start: Date,
        end: Date,
    }
}
----

[,json]
----
{
  "action": "stake",
  "token": {

}
