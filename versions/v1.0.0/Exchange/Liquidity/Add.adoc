===== Add

Add indicates an action of adding a token/tokens to liquidity pool.

===== Interface

[,typescript]
----
interface Add {
    action: string;
    tokens: {
        address?: string;
        value: string;
        name?: string;
        symbol?: string;
        decimals?: string;
        standard?: string;
    }[];
}
----
