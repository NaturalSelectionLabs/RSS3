===== Remove

Remove indicates an action of remove a token/tokens from a pool.

===== Interface

[,typescript]
----
interface Remove {
    action: string;
    tokens: {
        address?: string;
        value: string;
        name?: string;
        symbol?: string;
        decimals?: string;
        standard?: string;
    }[];
}
----
