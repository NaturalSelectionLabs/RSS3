==== Appoint

Proxy Appoint indicates that a user has delegated their operating rights to another user.

===== Interface

[,typescript]
----
interface Appoint {
    action: string
    profile: {
        profile_id: string
        address: string
        handle: string
        image_uri: string
        bio: string
        name: string
        expiry: string
        key: string
        value: string
    }
    proxy_address: string
}
----

===== Example

[,json]
----
{
    "action": "appoint",
