==== Share

Share indicates an action of sharing an existing post or comment, think of it as retweet.

===== Interface

[,typescript]
----
interface Share {
    handle?: string
    body?: string
    title?: string
    summary?: string
    media?: {
        address: string
        mime_type: string
    }[]
    tags?: string[]
    author_url?: string
    profile_id?: string
    publication_id?: string
    content_uri?: string
    timestamp?: number
    target?: {
        handle?: string
        body?: string
        title?: string
        summary?: string
        media?: {
            address: string
            mime_type: string
        }[]
        tags?: string[]
        author_url?: string
        profile_id?: string
        publication_id?: string
        content_uri?: string
    }
}
----
