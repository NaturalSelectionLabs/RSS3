===== Create

Profile Create indicates an action of creating new credential.

===== Interface

[,typescript]
----
interface Create {
    action: string
    profile_id: string
    address: string
    handle: string
    image_uri: string
    bio: string
    name: string
    expiry: string
    key: string
    value: string
}
----
