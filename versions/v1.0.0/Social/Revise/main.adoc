==== Revise

Revise indicates an action of updating an existing post.

[,typescript]
----
interface Revise {
  handle: string
  body: string
  title: string
  summary: string
  media: {
    address: string
    mime_type: string
  }[]
  tags: string[]
  author_url: string
  profile_id: string
  publication_id: string
  content_uri: string
  timestamp: number
}
----
